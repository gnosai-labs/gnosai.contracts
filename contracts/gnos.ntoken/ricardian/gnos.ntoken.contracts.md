# GNOS NFT

## create

Creates an NFT token definition with maximum supply, `nsymbol`, metadata URI, issuer, and IP owner.

## issue

Issues NFT units up to the created maximum supply. Issued units are tracked in `tokenstats` and owner-scoped `accounts`.

## retire

Retires issued NFT units from the issuer balance.

## transfer

Transfers one or more `nasset` NFT balances between accounts.
